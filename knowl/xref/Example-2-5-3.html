Example 2.69.

Add like terms.
  1. \(\displaystyle 5n+3n\)
  2. \(\displaystyle -4y-3y\)
  3. \(\displaystyle -6st+9st\)

Solution.

In each calculation, we do not change the variables we are adding; only the coefficient changes.
  1. \(\displaystyle 5n+3n = (5+3)n = 8n~~~~~~~~ \blert{\text{Add the coefficients,}~ 5 ~\text{and}~ 3.}\)
  2. \(\displaystyle -4y-3y = (-4-3)y = -7y~~~~~~~~ \blert{\text{Add the coefficients,}~ -4~ \text{and}~ -3}.\)
  3. \(\displaystyle -6st+9st = (-6+9)st = 3st~~~~~~~~ \blert{\text{Add the coefficients,}~ -6 ~\text{and} ~9.}\)
