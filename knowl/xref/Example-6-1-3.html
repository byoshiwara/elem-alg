Example 6.4.

Solve each equation without using a calculator.
  1. \(\displaystyle t^2=10,000\)
  2. \(\displaystyle b^2=\dfrac{4}{9}\)
  3. \(\displaystyle h^2=0.04\)
  4. \(\displaystyle m^2=(-17)^2\)
Solution.
For each equation, we find the square root of the right side. (That is, we find a number whose square gives the right side.) Remember that every positive number has two square roots.
  1. Because \(~~100^2=10,000,~~t=\pm 100\)
  2. Because \(~~\left(\dfrac{2}{3}\right)^2=\dfrac{4}{9},~~b=\pm \dfrac{2}{3}\)
  3. Because \(~~0.2^2=0.04,~~h=\pm 0.2\)
  4. Because \(~~17^2=(-17)^2,~~m=\pm 17\)
