Activity 3.10. Computing Slopes.

How do we know which two points to choose when we want to compute the slope of a line? It turns out that any two points on the line will do.
  1. Calculate the slope of the line \(~y=\dfrac{2}{3}x-2~\) shown in the figure:
    1. By using the points \(P(-3,-4)\) and \(Q(3,0)\text{.}\)
    2. By using the points \(R(6,2)\) and \(S(0,-2)\text{.}\)
    3. Do you get the same value for the slope in each case?
    1. Graph the line
      \begin{equation*} 4x-2y=8 \end{equation*}
      by finding the \(x\)- and \(y\)-intercepts.
      \(x\) \(y\)
      \(0\) \(\hphantom{0000}\)
      \(\hphantom{0000}\) \(0\)
    2. Compute the slope of the line using the \(x\)- and \(y\)-intercepts.
      \begin{equation*} m = \dfrac{\Delta y}{\Delta x} = \hphantom{0000} \end{equation*}
    3. Compute the slope of the line using the points \((4,4)\) and \((1,-2)\)
      \begin{equation*} m = \dfrac{\Delta y}{\Delta x} = \hphantom{0000} \end{equation*}
No matter which two points we use to calculate the slope of a line, we will always get the same result.
